# World Cup 2026 Takeover Pool

GitHub Pages app for tracking a World Cup pool.

## Files

- `index.html` - current live app
- `styles.css` - extracted stylesheet
- `data.js` - owner and team data
- `lines.js` - matchup data
- `bracket.js` - bracket module placeholder
- `app.js` - app module placeholder

## Workflow

Update matchup data before each new round, then refresh results from the app.
