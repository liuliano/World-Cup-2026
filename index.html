World Cup Pool Rules

  • Each owner starts with 3 teams.
  • Group-stage scoring: win = 3, draw = 1, loss = 0.
  • DraftKings/ESPN spreads are locked before each knockout round.
  • Spreads are regulation-time only.
  • If a favorite wins but does not cover, the underdog owner inherits the advancing team.
  • If a favorite advances in extra time or penalties, the underdog owner inherits the advancing team.
